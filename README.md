# CALCULATOR
A custom engineering calculator built with Python and PyQt5
This project is a custom-compiled Python desktop application that translates visual, human-readable math into secure, machine-level evaluations using an advanced regex parsing engine.

The fundamental crux lies in its "Safe Dictionary" architecture, which explicitly maps UI inputs to internal math libraries, completely eliminating the overlapping string-replacement bugs found in standard calculator scripts.

Engineered specifically for the rigorous, high-speed computational standards required in JEE Advanced and GATE CSE, it automatically handles implicit multiplication, fractional roots, and dynamic degree-to-radian conversions.

In terms of quality, it delivers a professional-grade user experience with multi-line word wrap, true cursor-based equation editing, and a crash-proof "silent fail" mechanism that protects your active workspace.

Structurally, it transcends a simple coding exercise by functioning as a fully packaged, installable Windows executable, complete with a high-contrast saffron UI, live history tracking, and your embedded personal signature.
